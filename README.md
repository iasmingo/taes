Este repositório é destinado ao projeto final da disciplina IF722 - TOPICOS AVANCADOS EM ENGENHARIA DE SOFTWARE.

## Geração incremental e verificação automatizada de diagramas UML assistidas por LLMs
A partir das limitações identificadas no artigo *How LLMs Aid in UML Modeling: An Exploratory Study with Novice Analysts* (Wang et al., 2024), sobretudo a falta de informações a respeito da consistência entre os três tipos de diagramas UML gerados — diagramas de casos de uso, de classes e de sequência — tendo como base um único estudo de caso, propomos um pipeline estruturado para geração e validação de modelos UML assistidos por LLMs.
